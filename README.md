# ♟️ MateVerse

A next-generation multiplayer chess platform with real-time gameplay,
tournaments, ELO ranking, spectating, and modern UI.

## Tech Stack
- Frontend: HTML, CSS, JavaScript (Vercel)
- Backend: Node.js, Express, Socket.IO
- Database: MongoDB
- Realtime: WebSockets

## Deployment
- Frontend → Vercel
- Backend → Render / Railway

## Features
- Multiplayer matchmaking
- Live spectating
- Tournaments & leaderboards
- Player profiles
- ELO rating system
